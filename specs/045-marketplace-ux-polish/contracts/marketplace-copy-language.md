# Contract: Marketplace Copy & Language

**Feature**: `045-marketplace-ux-polish`

## Product naming

| Context | Copy |
|---------|------|
| Hub embed title | `Marketplace` (not “Global Transfer Market”) |
| Hub subtitle / description | Sub-areas: Transfer Board · Scouting · Agent (· My Listings when P2P) |
| Slash command | Remains `/marketplace` |

## Navigation

| Control | Label |
|---------|-------|
| Back to hub from child | `Back to Market` |
| Back to My Listings | `Back to Listings` |

Emoji: optional on hub entry buttons; **no** required emoji on Back.

## Confirms

| Action | Confirm style intent |
|--------|----------------------|
| Buy / List | Success-styled confirm; Cancel secondary |
| Agent sale | Danger-styled confirm (irreversible sell) |
| Scout sign | Success-styled confirm |

## Shared errors

One ownership/session message family for “not your club / stale session” (single constant; variants only if genuinely different causes).

## Success

Short summary: what happened + coins delta/net when relevant — avoid walls of text.
